Constraints can be applied when creating a table, DROP the table and recreate it

CREATE TABLE student(
    student_id INT,
    name VARCHAR(20) ,
    major VARCHAR(20) DEFAULT='undecided' ,
    PRIMARY KEY(student_id)
);
