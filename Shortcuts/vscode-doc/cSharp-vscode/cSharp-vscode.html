Setup C# Extensions

Download .NET

01

Go to,
https://dotnet.microsoft.com/en-us/download/dotnet/8.0 and download '.NET 8.0'
'SDK 8.0' -> macOS -> x64

02

Open Vs code's termainl and check if you have installed .NET by typing, 

    dotnet --version

for version and 

    dotnet --info

for info

Install Externsions

03

Install the extension, 

    C# Dev kit

Create/Open Folder Start Project

04

Create/Open Folder in Vs code,

05

Start a project from template, press '⌘ + shift + p'
for the command palatte

  make sure the .NET package is enabled from the C# Dev Kit Extension, if so, open command palatte and select in Install .NET ... to finish install.

 Follow the prompts and finish .NET installation

06

After install shows complete in console, press '⌘ + shift + p' select
.NET start project
then select,
Console App
