The default color for outlines is black and the default color for filling in shapes is white.

To change the background color of the canvas, change the value of,

c.style.background = rgb(, , )

Colors can be adjusted with:

  • color names ex: 'blue' , 'green' , 'yellow' etc...
  • hex #0022ab
  • rgb(100,100,100,.4)

 To change the color of individual shapes add fillStyle = function before each shape function, and fill() after

//face
c.fillStyle =
c.ellipse(x, y, faceWidth, faceHeight, Math.PI / 4, 0, 2 * Math.PI)
c.fill()
c.stroke()
c.closePath()

//Eyes
c.fillStyle =
c.ellipse(x, y, faceWidth, faceHeight, Math.PI / 4, 0, 2 * Math.PI)
c.fill()
c.stroke()
c.closePath()

 *When making multiple ellipses, Remember to closePath() after each ellipse
