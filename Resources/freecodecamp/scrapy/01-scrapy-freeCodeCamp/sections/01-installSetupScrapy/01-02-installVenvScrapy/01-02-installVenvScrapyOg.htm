01-02-installVenvScrapy

Part 1 - venv

01

On mac python 3.9 and higher you don't need this, but on windows you might need to install virtualenv 

pip install virtualenv

02

Create and activate the venv

Part 2 - scrapy

03

Install scrapy into the venv with, 

pip install scrapy

Visit the site for the command if need be https://scrapy.org/

04

Run scrapy to see a list of its commands, you can see it say no active project. Create a project with the default folders in the next step. 

scrapy

05

Use the startproject command followed by the name of the project to create on in the current directory. 

scrapy startproject bookscraper
