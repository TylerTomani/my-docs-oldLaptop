CREATE TABLE student(
student_id INT AUTO_INCREMENT,
name VARCHAR(20),
major VARCHAR(20),
PRIMARY KEY(student_id)
);

02

We already have used 

SELECT * 
FROM student;

Which selects all the columns and rows from a table

03

Specify specific columns by placing their names after SELECT, Seperate them with commas 

    SELECT name , major
    FROM student;

04

You can prepend the columns with the specific table, (** this will come in handy when working with multiple tables) 

SELECT student.name, student.major
from student;
