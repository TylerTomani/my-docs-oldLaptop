08-04 Aggregation GROUP BY

01

'Aggregation' allows us to display the number of times and value a row of a column has.

The GROUP BY will group the number of times the value is counted, by whatever column argument we give it.

for example, find the number of male and female employees, 

SELECT COUNT(sex), sex
from employee
GROUP BY sex;

the count function counts the value and the sex column after the comma get displayed.
